A tool for generating Google Map Marker Icons in a sprite image sheet with a custom color marker and a range of alphanumeric symbols.

Do you need to generate custom marker Pins for Google Maps? This code takes the generic map pin and allows you to set a custom pin color and alphanumeric text on the pin. Rather than generating dozens or hundreds of separate images, this tool generates a single sprite PNG image. By using a sprite image, load performance is optimized when you have multiple pins on your map.
